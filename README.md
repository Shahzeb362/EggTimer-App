# 🥚 EggTimer App

The EggTimer App for boil eggs !  
Choose between **Soft, Medium, or Hard** modes, and the app will set the timer accordingly.

- 🔔 Plays an alert when the timer ends
- 🎨 Simple and easy-to-use interface
